Practice Test 1 Flash Cards

January 19, 2026 - 9 mins read

  1. What is a smart card?

    A smart card is a form of MFA that is in the shape of a credit card
    For MFA a smart card is an example of something you have. They store authentication on a physical device.

  2. What major cloud-related discipline includes a concept called Infrastructure as Code (IaC).

    DevOps
    DevOps is a major cloud-related discipline that emphasizes collaboration between development and operations teams to automate and streamline the software delivery process. IaC allows for the automated provisioning and management of infrastructure resources.

  3. What ports need to be open to allow access for SSH and HTTPS?

    22 for SSH (Secure Shell)
    443 for HTTPS (Hypertext Transfer Protocol Secure)
    SSH is used for secure remote access to VMs in the cloud
    HTTPS is used for secure web traffic

  4. What is the issue when a system on your VPC has an IP that starts with 169.254?

    Your DHCP service is not configured properly
    Devices will self assign an IP to them selves if a DHCP server does not give them one.
    They follow the pattern 169.254.x.x.

  5. What type of architecture does asynchronous communication between components and real-time responsiveness to system changes represent?

    Event-driven architecture
    In event-driven architecture changes are able to be reacted to in real time.

    With Asynchronous communication between components you can communicate and interact without waiting for responses.

    Real-time responsiveness to system changes means you immediately react to events without waiting for a specific request or command.

  6. What two deployment modes of VPN technologies are common in cloud environments?

    Point-to-Site VPNs
    Site-to-Site VPNs
    Point-to-Site VPNs are where devices connect securely to a cloud network.

    Site-toSite VPNs are a common deployment mode in modern cloud environments where a secure connection is established between two physical locations or networks.

  7. What is a P2V deployment

    Physical to virtual
    P2V involves migrating a physical server or machine to a virtual environment. This can help consolidate hardware resources, improve scalability, and reduce costs by running virtual instances of servers on a single physical machine.

  8. What’s the most common parameter used to create a tiered storage approach in the cloud?

    Frequency of access
    The frequency of access is the most common parameter used to create a tiered storage approach in the cloud. Storage can be tiered to ensure that frequently accessed data is stored in faster storage based off of how often it is accessed.

  9. What is serverless computing?

    When resources are made available in the cloud without the need of "standing up" and mainting virtual machines.
    Serverless Computing allows developers to run code without the need to manage servers or infrastructure. Resources are on-demand.

  10. What is the primary function of service discovery in cloud-native environments?

    To automatically locate and connect services within a network.
    Services dynamically discover and communicate with each other without the need for manual configuration. This enables flexibility and scalibility in cloud-native applications.

  11. What is the term for support of sudden increase in capacity needs?

    Bursting
    Bursting in cloud computing refers to the ability of the cloud to quickly and seamlessly scale up resources to meet sudden increases in capacity needs.

  12. What type of cloud solution deployment consists of two fully functional solution environments?

    Blue-Green Deployment
    Blue-Green Deployment is a deployment strategy where two fully functional solution environments are maintained simultaneously.

  13. What approach is used in the LAN for assigning a data frame with its VLAN?

    Tagging
    Assigns a data frame with its VLAN. VLAN tagging adds a VLAN identifier to the ethernet frame, allowing switches to identify and forward the frame to the correct VLAN.

  14. What would the requirement be called to be able to restore all opperations to exactly one previous 24-hour period?

    RPO (Recovery Point Objective)
    The RPO is the acceptable amount of data loss in case of a disaster or system failure. In this case your organization wants you to be able to restore operations to exactly 24-hours ago.

  15. What storage technology permits the convergence of LAN and SAN traffic?

    FCoE (Fibre Channel over Ethernet)
    FCoE is a storage technology that allows Fibre Channel traffic to be transmitted over Ethernet networks. It enables the convergence of LAN and SAN traffic by allowing Fibre Channel storage traffic to travel over Ethernet networks, simplifying network infrastructure and reducing costs.

  16. In the shared security model, who is responsible for securing the virtualization software in a public cloud environment?

    The CSP(Cloud Service Provider)
    The CSP is responsible for securing the underlying virualizaiton software in a public cloud environment. This includes ensuring the hypervisor and other virtualization components are properly configured, patched, and protected from security threats.

  17. What is Object storage good for in the cloud?

    Storing large volumes of unstructured data such as media files and backups
    Object storage is good for large volumes of unstructured data at low costs. It's also more easily scaled and doesn't requre a hierachical file system. Objects are stored with metadata, making it easier to manage and access large amounts of data efficiently.

  18. Where would you IaaS

    You would use IaaS when you want to manage the guest OS and images on provider managed compute
    IaaS provides virtualized compute where you control the guest OS while the provider manages the underlying infrastructure.

  19. What modern network virtualization technology addresses issues with namespace size and layer 3 routhing support

    Virtual Extensible Local Area Networks (VXLANs)
    Virtual Extensible Local Area Networks (VxLANs) address issues with namespace size and layer 3 routing support by providing a way to extend Layer 2 networks over Layer 3 networks. VXlANs use encapsulation techniques to create virtual networks that can scale to accommodate larger network sizes and support layer 3 routing, making them an ideal solution for modern network virtualization challenges.

  20. What does metadata provide in cloud storage?

    Metadata provides descriptive information about data, such as creation date and permissions.
    Metadata provides descriptive data such as creation date, permissions, file type, size, and other attributes.

  21. What virtual machine concept permits the restoration of a server to a specific moment in time?

    Snapshot
    Snapshots are a virtual machine concept that allows the restoration of a server to a specific moment in time by capturing the current state of the virtual machine, including its disk, memory, and settings.

  22. What is the primary function of the ELK Stack in cloud monitoring and logging.

    To collect, store, search and visualize log data from multiple sources.
    self explanitory.

  23. What two reasons might an organization have multiple VPCs in their own private cloud infrastructure?

    Public/private access Prod/Dev environments
    Multiple VPCs may be used to esablish public and private access control for resources. They may also be used to provide a produciton and development environment. This helps in seperating critical production systems from the development and testing environments. This reduces risk of accidental changes.

  24. What type of cloud deployment represents a partnership between multiple different organizations?

    Community Cloud
    Community cloud deployment involves a partnership between multiple organizations that have shared intersts, such as regulatory concerns or security requirements.

  25. What is the main issue with security in SNMP version 2c

    The password information is sent in clear text.
    The main issue with security in SNMP version 2c is that password information is sent in clear text, making it vulnerable to interception and unauthorized access.

  26. What is often considered the ultimate solution for highly available cloud-services

    Clustered servers
    Clustered servers are often considered the ultimate solution for highly available cloud services because they provide redundancy and load balancing. Multiple servers can continue operating by distributing the workload.

  27. What two common benefits occur when implementing a cloud resource tagging strategy?

    Improved cost allocation and racking Simplified automation and resource management.
    Cost allocation uses tagging to watch resources based on their usage, department, or project. This helps in tracking expenses and optimizing resource utilization effectively. Simplified automation and resource management provides a structured way to identify and classify resources.

  28. What is a frequent system and platform concern when considering a cloud deployment for existing server resources

    64-bit vs 32-bit
    It is crucial to ensure compatibility and performance optimization when transitioning existing server resources to the cloud environment.

  29. What security technology might feature signature-based attack prevention?

    IPS (Intrusion Prevention System)
    IPS is a security technology that can feature signature-based attack prevention. IPS monitors network traffic for malicious activities or security policy violations and can block or prevent these attacks based on predefined signatures of known threats.
